# Quiz Game 🎮

A simple command-line quiz game built with Python.

## Features

- Multiple quiz questions
- Score tracking
- Easy-to-use terminal interface

## Technologies Used

- Python 3

## How to Run

1. Clone the repository

```bash
git clone https://github.com/build-with-vishnu/quiz-game.git
```

2. Navigate to the project

```bash
cd quiz-game
```

3. Run the program

```bash
python quiz.py
```
